# Assets Directory

## Icons (`/icons`)

**Status**: Pending design resources

To add animated icons:
1. Create SVG icons for common UI actions
2. Add animation CSS in global.css
3. Import and use in components

Recommended icons:
- Loading spinner
- Success/error indicators
- Game-specific icons (detective, clue, etc.)

## Images (`/images`)

**Status**: Pending design resources  

To add optimized images:
1. Add game screenshots/previews
2. Add placeholder images for empty states
3. Optimize with tools like ImageOptim or Squoosh
4. Consider WebP format for better compression

Recommended images:
- Game type preview images
- Empty state illustrations
- Background patterns
- Logo and branding assets

## Notes

Tasks T119 and T120 are optional and require design resources. The platform is fully functional without these assets - they primarily enhance visual appeal.
